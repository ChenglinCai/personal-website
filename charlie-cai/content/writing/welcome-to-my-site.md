## Writing in Markdown

Markdown is a simple way to format text. A few examples:

- **Bold** text uses `**double asterisks**`
- *Italic* text uses `*single asterisks*`
- Links look like [this](https://example.com)

You can also write lists, quotes, and headings without touching any code.

> A blockquote looks like this — handy for pulling out a key line.

### Code, too

If you ever want to show code, fenced blocks work:

```js
console.log("Hello from a code block");
```

That's the whole idea: you focus on the words, and the site handles the styling.
